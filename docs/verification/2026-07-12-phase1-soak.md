# Phase 1 render-soak evidence — 2026-07-12

The Debug `AetherStudio` build at foundation commit `73778de` opened the portable
`tests/fixtures/triangle.aetherproject` fixture and rendered continuously for 30 minutes. The
monitor checked the process every 30 seconds and exited successfully only after all 60 checks.

- Process: `AetherStudio` PID 89394
- Initial RSS: 84,752 KiB
- Final RSS: 135,712 KiB
- Final physical footprint reported by `leaks`: 121.9 MiB; peak 133.9 MiB
- Metal and Swift test suites: passing before and after the run

The post-soak `leaks` scan reported 14,400 bytes in 288 objects, all shown under three
`LNDaemonApplicationInterface`/AppIntents XPC cycles. It did not attribute a leak to AETHER's C++,
Metal resources, bridge, or frame loop. macOS also warned that the live process was not fully
debuggable, so this result is evidence, not a claim of a mathematically zero-leak process.

The Phase 1 exit item remains open until an API-validation GPU capture confirms the named passes and
resources. The soak itself is complete.
